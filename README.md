# 项目说明 <br>
2026.4.10<br>

### 项目名称：<br>
Digital Cleansing/数据清道夫<br>

### 项目简介<br>
《数据清道夫》/ Digital Cleansing是一款由DoorBell Labs制作组制作的基于网格的2D动作游戏。<br>
玩家扮演一个超级数据网络中的杀毒软件，深入网络内部，破坏防火墙以深入网络核心，最终击败藏匿于根服务器的病毒母体，并顺手消灭路上的病毒。
游戏使用Unity引擎开发（版本号2022.3.62f3c1）。<br>

---

### 项目结构<br>
使用Unity自动生成的项目文件。<br>
添加的素材存储于Assets文件夹。其中脚本、音频、美术资源分别存放于Assets下的Scripts, BGM和Sprites文件夹中。<br>

### 使用说明<br>
项目支持Windows操作系统。<br>
下载仓库中DigitalCleansing.zip压缩包到本地，解压后运行其中的Digital Cleansing.exe即可游玩。
也可将仓库全部内容下载到本地，并用对应版本Unity Editor打开文件夹。<br>
游玩时请切换至英文输入法。<br>

---

### 玩法机制<br>
游戏为半线性流程，包含简短的教程、大厅、3条支线（目前每条支线含3小关）以及最终的Boss战。小关内玩法包括推箱子解谜、躲避激光同时破坏目标、与敌人战斗等。预计初见通关时长约15-30分钟，随操作熟练度高低可能有较大差异。<br>
教程结束后传送至大厅，玩家可随意选择三条支线中的一条进入，死亡会回到大厅，支线通关后此支线关闭。3条支线均通关后开启Boss战传送门。<br>
每一关卡内，玩家在正方形地砖组成的网格地图上移动并踢击敌人、箱子等其他实体以达成目标。小关内的目标是破坏所有的防火墙和敌人，并进入前往下一关的传送门。<br>
游戏开始的教程中有简要的玩法介绍。<br>

下面是一些主要游戏对象的介绍：<br>
    地图：由正方形瓷砖组成的可移动网格，拥有多种贴图。<br>
    激光：场上按一定规律生成的激光，玩家被激光攻击会掉血。<br>
    预警：在即将被激光或敌人攻击的瓷砖上生成的预警特效。<br>
    敌人：包括向四周随机空格跳动的近战敌人和向空格方向发射穿透激光的远程敌人，按照音乐节拍周期性行动。可被玩家踢动，两个敌人被玩家踢动相撞时二者均会死亡。<br>
    箱子：不可破坏的实体，箱子后方为空格时可被玩家踢动。<br>
    防火墙：不可移动的实体，被玩家多次攻击后碎裂。<br>
    传送门：地图中的实体，场上没有防火墙和敌人时激活，玩家攻击激活的传送门即可前往下一关。<br>
    Boss：占据3*3格的实体，共3个阶段30点HP，有召唤敌人、生成多种模式激光、狂暴、带预警的范围秒杀等技能。将场上小怪踢到Boss身上以造成伤害，击杀Boss会生成通往胜利界面的传送门。<br>

### 操作方法：<br>
  移动/攻击 ———— WASD/方向键<br>
  切换至暂停界面 ———— Esc键<br>
  重开当前关卡 ———— R<br>
  重开至上一节点 ———— L<br>
  当前关卡开启无敌模式 ———— I<br>
  界面交互 ———— 鼠标左键单击<br>

### 项目特色<br>
  原创的美术资源：项目使用的所有美术资源除字体外均为原创。<br>
  富有节奏感的战斗：在战斗关卡内播放紧张的bgm（来自开源社区），角色动画、敌人行动和激光生成完全基于音乐节拍。<br>
  良好的操作手感：操作键位简单，角色移动较为灵活，且与动画精准配合。<br>
  自由的关卡编辑：游戏物体和脚本耦合度较低，在Unity Editor中可以通过较为简单的操作实现关卡的自由编辑和添加。<br>

### 创意来源<br>
  游戏的主要玩法借鉴了《地狱把妹王》(Helltaker),《节奏地牢》(Crypt of the Necrodancer),《陷阵之志》(Into the Breach)等游戏。<br>
